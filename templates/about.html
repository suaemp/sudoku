ABOUT M&W SUDOKU



This sudoku game implementation was created by Michał Rolski and Wiktoria Śladowska (M&W) as JavaScript learning excercise. Game uses postgreSQL database as backend data storage as well as Flask module for backend server.



Sudoku game rules:

  1. Only use the numbers 1 to 9,
  2. Avoid trying to guess the solution to the puzzle,
  3. Only use each number once in each row, column, & grid,
  4. Use the process of elimination as a tactic,
  5. Use cross-hatching and penciling in techniques.
  6. Each board has only ONE possible solution!
